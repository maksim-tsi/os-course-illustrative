# os-course-illustrative
Illustrative examples for the private course on Operating Systems and Systems Programming

## Repository Structure

- `class-lab-02-sigterm`: AI Worker Process Manager (SIGTERM vs SIGKILL)
- `class-lab-03-zerocopy`: High-Performance Zero-Copy AI Pipeline
- `lecture5_kernel_objects`: Examples of IPC (Pipes, Shared Memory) using C.
